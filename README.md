# practical-scheme

Hands-on exercises and projects in **Scheme** (a dialect of Lisp), completed as part of my Computer Science and Software Engineering studies at **CFI-CIRAS** (Brazzaville, Congo).

This repository focuses on applying Scheme to real problems — not just syntax, but logic, recursion, and functional thinking.

---

## 📁 Repository Structure

```
practical-scheme/
│
├── basics/             # Core syntax, expressions, and data types
├── conditionals/       # if, cond, when, unless
├── functions/          # Lambda, recursion, higher-order functions
├── logic/              # Boolean operators and lazy evaluation
├── projects/           # Practical scenarios and mini-projects
└── ...                 # More to come
```

---

## 🧠 Concepts Covered

- **Expressions & evaluation** — how Scheme reads and executes code
- **Conditionals** — `if`, `cond`, `when`, `unless`
- **Logical operators** — `and`, `or`, `not` with lazy evaluation
- **Recursion** — solving problems through self-referencing functions
- **Higher-order functions** — functions that take or return functions
- **Functional programming** — immutability, pure functions, composability
- *And more as the learning progresses...*

---

## 🎯 Why Scheme?

Scheme is a minimal, elegant language that sharpens logical and algorithmic thinking. Mastering it builds a strong foundation for understanding any programming language and paradigm — especially functional patterns now common in modern JavaScript, Python, and beyond.
